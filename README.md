# Анализ музыкальных треков Spotify

Проект направлен на исследование музыкального каталога Spotify: популярность исполнителей, длительность композиций, распределение популярности и взаимосвязь между параметрами.

## Используемые технологии:
- Python (pandas)
- Jupyter Notebook

## Задачи исследования:
- провести обзор данных
- определить самых популярных артистов
- выявить топ релизов по популярности
- изучить связь популярности артиста и его треков
- анализ длительности треков

## Основные выводы:
- В датасете 8582 трека, данные почти чистые.
- Средняя продолжительность трека ≈ 3.5 мин.
- Популярность артистов и треков связана положительно.
- Топ артистов и топ по подписчикам - разные выборки.
- Выявлены самые короткие/длинные треки и популярные композиции.
